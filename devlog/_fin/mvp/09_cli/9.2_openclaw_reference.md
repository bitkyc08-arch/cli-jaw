# 9.2 레퍼런스 목록 — OpenClaw 명령어

> `openclaw-ref/src/auto-reply/commands-registry.data.ts` 기준.
> cli-claw 명령어 구현 시 참조용.

---

## Status (상태 확인)

| 명령어            | 별칭      | 설명                    |
| ----------------- | --------- | ----------------------- |
| `/help`           |           | 사용 가능한 명령어 보기 |
| `/commands`       |           | 모든 슬래시 명령어 나열 |
| `/status`         |           | 현재 상태 표시          |
| `/whoami`         | `/id`     | 내 sender id 표시       |
| `/context`        |           | 컨텍스트 구성/사용 설명 |
| `/export-session` | `/export` | 세션을 HTML로 내보내기  |

## Session (세션 관리)

| 명령어     | 인자                  | 설명               |
| ---------- | --------------------- | ------------------ |
| `/session` | `ttl <duration\|off>` | 세션 TTL 설정      |
| `/reset`   |                       | 현재 세션 초기화   |
| `/new`     |                       | 새 세션 시작       |
| `/compact` | `[instructions]`      | 세션 컨텍스트 압축 |
| `/stop`    |                       | 현재 실행 중지     |

## Options (옵션 설정)

| 명령어       | 별칭              | 인자                                     | 설명             |
| ------------ | ----------------- | ---------------------------------------- | ---------------- |
| `/model`     |                   | `<provider/model>`                       | 모델 설정        |
| `/models`    |                   | `[provider]`                             | 모델 목록        |
| `/think`     | `/thinking`, `/t` | `off\|minimal\|low\|medium\|high\|xhigh` | thinking level   |
| `/verbose`   | `/v`              | `on\|off`                                | verbose 모드     |
| `/reasoning` | `/reason`         | `on\|off\|stream`                        | reasoning 표시   |
| `/elevated`  | `/elev`           | `on\|off\|ask\|full`                     | elevated 모드    |
| `/usage`     |                   | `off\|tokens\|full\|cost`                | usage 표시       |
| `/exec`      |                   | `host security ask node`                 | exec 기본값 설정 |
| `/queue`     |                   | `mode debounce cap drop`                 | 큐 설정          |

## Management (관리)

| 명령어        | 별칭    | 인자                                        | 설명                   |
| ------------- | ------- | ------------------------------------------- | ---------------------- |
| `/subagents`  |         | `list\|kill\|log\|info\|send\|steer\|spawn` | 서브에이전트 관리      |
| `/focus`      |         | `<target>`                                  | 스레드 → 세션 바인딩   |
| `/unfocus`    |         |                                             | 바인딩 해제            |
| `/agents`     |         |                                             | 바인딩된 에이전트 목록 |
| `/kill`       |         | `<target\|all>`                             | 서브에이전트 종료      |
| `/steer`      | `/tell` | `<target> <message>`                        | 서브에이전트 가이던스  |
| `/approve`    |         |                                             | exec 요청 승인/거부    |
| `/allowlist`  |         |                                             | 허용목록 관리          |
| `/config`     |         | `show\|get\|set\|unset`                     | 설정값 관리            |
| `/debug`      |         | `show\|reset\|set\|unset`                   | 디버그 오버라이드      |
| `/activation` |         | `mention\|always`                           | 그룹 활성화 모드       |
| `/send`       |         | `on\|off\|inherit`                          | 전송 정책              |

## Tools (도구)

| 명령어     | 인자             | 설명                         |
| ---------- | ---------------- | ---------------------------- |
| `/skill`   | `<name> [input]` | 스킬 실행                    |
| `/bash`    | `<command>`      | 호스트 셸 명령어 (text 전용) |
| `/restart` |                  | OpenClaw 재시작              |

## Media

| 명령어 | 인자                                                     | 설명     |
| ------ | -------------------------------------------------------- | -------- |
| `/tts` | `on\|off\|status\|provider\|limit\|summary\|audio\|help` | TTS 제어 |

## Docks (채널 연동)

채널 독 플러그인 등록 시 `/dock-{id}` 형태로 동적 생성.
